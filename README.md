# Natural Disaster Classification using Deep Learning

Proyek ini adalah tugas kelompok mata kuliah Deep Learning untuk mengklasifikasikan 4 jenis bencana alam (Earthquake, Land_Slide, Urban_Fire, Water_Disaster) berdasarkan gambar.

## Ringkasan Proyek
Kami membangun dan membandingkan model Convolutional Neural Network (CNN) yang dibangun dari awal (*scratch*) dengan model *Transfer Learning* menggunakan **EfficientNetB0**. 

## Apa yang ada di dalam repository ini?
* `PROJECT_DL_FINAL.ipynb` : Berisi seluruh kode Python, mulai dari pemrosesan data (EDA), pembuatan model, pelatihan, hingga evaluasi.
* `REPORT PROJECT ASSIGNMENT DEEP LEARNING.pdf` : Laporan lengkap yang membahas latar belakang, metodologi, dan analisis hasil secara mendalam.
